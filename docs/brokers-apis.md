# summary table of the best brokers supporting automated trading like alpaca and also the best financial data api liek alphavantage and finnhub

Here is a compact summary table of widely used brokers for automated trading (Alpaca-style) and leading financial data APIs (Alpha Vantage / Finnhub–style).[^1][^2][^3][^4][^5][^6]

## Brokers with strong APIs

| Broker | Main Assets | Notes on Automation/API |
| :-- | :-- | :-- |
| Alpaca | US stocks, ETFs, crypto | Dev-first REST/WebSocket API, paper trading, good docs, commission-free equities[^7][^4]. |
| Interactive Brokers | Global stocks, options, futures, FX | Very full-featured APIs (TWS, Web, FIX), broad markets, institutional-grade access[^2][^8]. |
| Tradier | US stocks, options | Cloud brokerage with REST API focused on algo trading and embedded brokerage[^9][^4]. |
| OANDA | FX, CFDs | Well-documented REST API for FX/CFD algos and integrations (often via MT4/MT5 too)[^10]. |
| Exness / Octa / XM | FX, CFDs | Retail FX/CFD brokers with stable APIs and MT4/MT5/cTrader support for automation[^11]. |
| Deriv | Multi-asset CFDs | Proprietary API for programmatic trading plus data access in multiple languages[^11]. |

## Financial data APIs

| API Provider | Key Coverage | Best Use Cases / Notes |
| :-- | :-- | :-- |
| Alpha Vantage | Stocks, FX, crypto, fundamentals, news | Real-time + long historical OHLCV, indicators, news \& sentiment; generous free tier[^2][^3]. |
| Finnhub | Global equities, FX, macro, alt data | Real-time prices, earnings, insider trades, sentiment; strong for research and AI[^3][^6]. |
| IEX Cloud | US equities | Real-time \& historical prices, fundamentals, news; good docs and production SLAs[^3]. |
| Marketstack | Global stocks | Real-time \& historical EOD/intraday, simple pricing; solid for dashboards and apps[^3]. |
| Twelve Data | Stocks, FX, crypto, commodities | Multi-asset feed with indicators and news; flexible plans for startups[^3]. |
| Financial Modeling Prep | Fundamentals, statements, prices | Detailed financials, ratios, screening data; good for valuation tooling[^12][^5]. |

If you share your target region (US-only vs global), assets (equities vs options/futures/FX/crypto), and budget constraints, a more opinionated “pick this stack” recommendation can be tailored.
