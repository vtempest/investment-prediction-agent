# API Reference

Welcome to the Investment Prediction Agent API documentation. This API provides comprehensive access to stock market data, AI-powered trading analysis, backtesting capabilities, and portfolio management.

## API Overview

The Investment Prediction Agent API is organized around REST principles. Our API has predictable resource-oriented URLs, accepts form-encoded request bodies, returns JSON-encoded responses, and uses standard HTTP response codes, authentication, and verbs.

## Base URLs

- **Production**: `https://autoinvestment.broker/api`
- **Development**: `http://localhost:3000/api`

## API Categories

### Stocks
Access real-time and historical stock market data, including quotes, gainers, sectors, and company search.

### Trading Agents
Leverage AI-powered trading analysis using debate agents and groq-powered analysis for investment decisions.

### Backtesting
Test trading strategies against historical data to evaluate performance before deploying in live markets.

### SEC Filings
Retrieve and analyze SEC company filings and regulatory documents.

### User Portfolio
Manage your investment portfolio, track positions, and monitor performance.

### User Strategies
Create, update, and manage custom trading strategies and algorithmic scripts.

### User Signals
Configure watchlists and receive trading signals based on your strategies.

### Statistics
Access statistical analysis and predictive modeling for market data.

## Getting Started

Browse the endpoints in the sidebar to explore the full API capabilities. Each endpoint includes:

- Request parameters and body schemas
- Response schemas and examples
- Authentication requirements
- HTTP methods and status codes

## Authentication

Most endpoints require authentication. Include your API credentials in the request headers as specified in each endpoint's documentation.
